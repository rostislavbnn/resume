# 📊 Markov Chains & SVD Analysis

## 📝 Описание
Учебный проект по математическому моделированию и линейной алгебре. Включает два задания:
1. Прогнозирование распределения исследователей (промышленность / наука) с использованием цепей Маркова.
2. Реализация и применение сингулярного разложения (SVD) для анализа данных.

## 🎯 Задачи

### Часть 1: Моделирование цепью Маркова
- **Условие:** В 2024 году 30% исследователей работают в промышленности, 70% — в науке. Матрица переходов фиксирована.
- **Цель:** Спрогнозировать распределение на 100 лет вперёд (2024–2124) и визуализировать динамику.
- **Что сделано:**
  - Реализовано умножение вектора состояния на матрицу переходов
  - Построен график изменения долей за 100 лет
  - Проанализирована сходимость к стационарному распределению

### Часть 2: Сингулярное разложение (SVD)
- **Цель:** Применить SVD для снижения размерности / анализа структуры данных.
- **Что сделано:**
  - Реализовано SVD через `numpy.linalg.svd`
  - Проанализированы сингулярные значения и векторы
  - Визуализированы результаты разложения

## 🛠 Стек
- Python 3.9+
- NumPy (матричные операции, SVD)
- Matplotlib (визуализация динамики)
- Jupyter Notebook

# 🏥 EDA: Факторы риска диабета (Медицинские данные)

## 📝 Задача
Провести разведочный анализ медицинских данных для выявления закономерностей, связанных с диабетом, и оценить риск возникновения заболевания.

## 🔍 Базовый анализ
- **Возраст:** Диабет преимущественно встречается у взрослых 40+
- **Глюкоза:** Значительно выше у диабетиков (ключевой маркер)
- **ИМТ:** Наблюдается слабый положительный тренд при диабете
- **Гемоглобин:** Повышен у диабетиков, но перекрывается с нормой здоровых людей (5.5-7.0)
- **Пол:** Распространённость одинакова среди мужчин и женщин
- **Коморбидность:** У диабетиков чаще встречается гипертония

## 🔬 PRO-этап: Углублённый анализ
- **Сегментация выборок:** сравнение распределений внутри групп (по полу, возрасту, ИМТ)
- **Pairplot с hue='diabetes':** визуальный поиск кластеров и разделимости классов
- **Тепловая карта корреляций (Phik):** оценка нелинейных зависимостей между признаками
- **Дополнительные инсайты:**
  - Комбинация «возраст 40+ + глюкоза >120 + гипертония» значительно повышает вероятность диабета
  - ИМТ сам по себе — слабый предиктор, но в сочетании с глюкозой усиливает сигнал

## 🛠 Стек
- Python, Pandas, NumPy
- Seaborn (pairplot, heatmap)
- Phik (корреляции для категориальных и смешанных данных)
- Scikit-learn (масштабирование, кодирование)

- **Сравнительный анализ:** медианные траты по категориям товаров для каждого сегмента

### Влияние семейного статуса

# 💾 SQL Queries Portfolio

## 📝 Описание
Коллекция продвинутых SQL-запросов для бизнес-аналитики. Включает работу с представлениями, триггерами, хранимыми процедурами и курсорами. Основной фокус — финансовая логика и отчётность.

## 🎯 Что внутри

### 📊 Представления (VIEWS)
- `SalesView` — отчёт по продажам с расчётом цены в рублях (валюта + курс + налог)
- `CustomerMetrics` — агрегированные метрики по клиентам (LTV, средний чек, частота покупок)
- `ProductPerformance` — эффективность товаров по периодам

### ⚡ Триггеры (TRIGGERS)
- Автоматическое обновление итоговых сумм при изменении заказов
- Валидация данных перед вставкой (проверка диапазонов, форматов)
- `INSTEAD OF` триггеры для представлений

### 🔄 Курсоры (CURSORS)
- Построчная обработка сложных бизнес-кейсов
- Генерация отчётности с кастомной логикой расчётов

### 📦 Хранимые процедуры (STORED PROCEDURES)
- Автоматизация ежемесячной отчётности
- Расчёт бонусов менеджерам по итогам периода

### 🗄️ Проектирование БД
- Нормализация до 3НФ
- ER-диаграммы, связи между таблицами
- Индексы для оптимизации запросов

# 🐞 Bug Chase Simulation — Задача о преследовании жуков

## 📝 Описание
Интерактивная визуализация классической математической задачи о N жуках. Каждый жук движется к следующему по вершинам правильного многоугольника, образуя спиральные траектории схождения к центру.

## 🎯 Математическая постановка задачи
- N жуков расположены в вершинах правильного N-угольника
- Каждый жук движется к следующему жуку (по часовой стрелке) с постоянной скоростью
- Траектории образуют логарифмические спирали
- Все жуки встречаются в центре многоугольника

## 🛠 Стек
- Python 3.9+
- Tkinter (GUI, анимация, интерактивные элементы)
- Math (тригонометрия, вычисление расстояний)

# 🎯 Monte Carlo Pi Simulation

## 📝 Описание
Интерактивная визуализация метода Монте-Карло для приближённого вычисления числа π. Демонстрирует принцип статистического моделирования через генерацию случайных точек и оценку их попадания в область.

## 🎯 Математическая основа

### Принцип метода
- Квадрат со стороной 1 и вписанный круг радиусом 0.5
- Генерируются N случайных точек в квадрате
- Отношение точек внутри круга к общему числу точек ≈ π/4
- **Формула:** `π ≈ 4 × (точки_внутри / точки_всего)`

### Точность оценки
| Точек (N) | Ожидаемая точность | Время сходимости |
| :--- | :--- | :--- |
| 100 | ~1 знак после запятой | Мгновенно |
| 1,000 | ~1-2 знака | Мгновенно |
| 10,000 | ~2-3 знака | < 1 секунды |
| 100,000 | ~3-4 знака | 1-2 секунды |
| 1,000,000 | ~4-5 знаков | 5-10 секунд |

## 🛠 Стек
- Python 3.9+
- NumPy (векторизованные вычисления, генерация случайных чисел)
- Matplotlib (визуализация точек и окружности)
- Tkinter (GUI, интерактивные элементы)
- FigureCanvasTkAgg (интеграция Matplotlib в Tkinter)
